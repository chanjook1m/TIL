# CSS Variables (Custom properties)

css 에서 사용하는 값 (e.g. color code)을 변수로 만들어서 사용하는 방법이다.

개발을 할때 중복되는 값을 변수로 사용하는데 css에서도 이러한 것이 가능하다.

# 언제?

  • 중복되는 값을 변수로 만들어서 값을 한번에 수정하려고 할때

# 예제

  1. 기본 사용법
element {
  --bg-color : red;
}
  1. 일반적인 사용법

임의의 클래스인 :root 를 element로 사용한 후 적용하고 싶은 element에서 var(--변수명) 형태로

사용한다.

:root {
	--bg-color: red;
  --bg-color2: yellow;
}

body {
	background-color: var(--bg-color)
}
  1. 변수가 정의 되지 않았을 경우

,를 사용해 정의 되지 않았을때 적용할 속성을 넣으면 된다

body {
	background-color: var(--bg-color, blue)
}

이때 다른 변수를 대체 속성으로 넣으려면 아래와 같이 사용해야 한다

body {
background-color: var(--bg-color, var(--bg-color2, blue))
}
  1. 변수 값이 잘못된 값일 경우

예를 들어 background-color에 px값이 들어갈 경우 css에서는 이러한 상황에서는 일반적으로

기본 설정된 값이나 상속된 즉, parent 요소의 값을 사용한다.

  1. 자바스크립트에서 css variable을 사용하려면
element.style.getPropertyValue("--변수명") // 값 가져오기
element.style.setProperty("--변수명", "값") // 값 설정하기
