# appearance

# 언제?

  1. 태그에 기본적으로 적용되는 특정 랜더링 엔진의 스타일을 태그에 적용하고 싶을때
  2. 태그에 기본적으로 적용되는 특정 랜더링 엔진의 스타일을 태그에서 제거하고 싶을때
  • 여기서 랜더링 엔진은 webkit (크롬, 사파리 등)이나 gecko (파이어폭스) 같은 것을 말한다.
