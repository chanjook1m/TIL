# Box-sizing

# 언제?

html 태그의 padding과 border 속성을 width 그리고 height 속성에 포함시킨다. 간단하게 말하면 해당 태그의 padding과 border 속성이 무시된다고 생각하면 된다.

# 예제

Codepen

위의 예제에서 withborderBox1이 withBorderBox2보다 약간 더 긴것으로 보인다.

이유는 withborderBox1 에서는 border 속성이 무시되지 않고 border 길이가 1px 이기 때문에

width 200px에 2px이 더해져 202px이 되기 때문이다.
