# Px, Em, Rem

이 세 단위 모두 css에서 사이즈를 지정할때 사용되는 단위이다.

Px은 기준이 자기 자신, Em은 부모 태그 그리고 Rem은 html 태그의 기본 설정을 참조 한다.

# 언제?

유저는 다양한 크기의 핸드폰, 다양한 브라우저 등을 사용하기에 사용환경이 매우 다르다.

따라서 접근성 기준에서는 px 보다는 상대적 크기를 지정하는 em, rem을 지향하는 것이 좋다고

한다.

# 예제

font-size를 가지고 예를 들어보자.

Codepen

  1. Em

부모 태그의 font-size가 20px인 경우 1em=20px이 된다. 따라서 1.5em=30px이다.

위의 예제에서 html의 font-size값을 아무리 바꿔봐도 em1 태그의 font-size는 바뀌지 않는다.

  1. Rem

html 태그의 font-size가 20px인 경우 1rem=20px이 된다.

참고로 html 태그의 기본 font-size는 16px이다. 그렇기에 일반적으로 html 태그에

font-size: 62.5% 값을 준다. 이유는 16의 62.5%가 10px이고 따라서 1rem이 10px이 되어 계산하기가

편해진다.

  1. Media Query

Media Query를 사용할때는 일반적으로 em을 사용해야 접근성이 높아진다고 한다.
